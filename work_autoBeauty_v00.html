Auto Beauty

I wrote this tool for a compositing software (Nuke) with the goal to accelerate the workflow of visual effects artists.
It allows the user to import a multi render pass from render engine like Cycle from Blender) or Redshift and reconstruct to match the beauty. Some options like denoise, sharpen, and grade are directly implemented in the menu.
